<Exercise n="11.1" title="Indovina il tipo">

Stampa il tipo di ciascun valore con `type()`.

```py live
print(type(42))
print(type(3.14))
# aggiungi una stringa e un booleano
```

<Solution>

```py live readonly
print(type(42))
print(type(3.14))
print(type("ciao"))
print(type(True))
```

</Solution>

</Exercise>

<Exercise n="11.2" title="Conversioni">

Converti la stringa `"7"` in intero e sommala a `3`.

```py live
s = "7"
# converti e somma
```

<Solution>

```py live readonly
s = "7"
print(int(s) + 3)
```

</Solution>

</Exercise>
