<Exercise n="10.1" title="Scambia due variabili">

Hai due variabili `a` e `b`. Scambia i loro valori, poi stampali.

```py live
a = 3
b = 8
# scambia a e b, poi stampali
```

<Solution>

```py live readonly
a = 3
b = 8
a, b = b, a
print(a, b)
```

</Solution>

</Exercise>

<Exercise n="10.2" title="Calcola la media">

Date tre variabili numeriche, calcola e stampa la loro media aritmetica.

```py live
a = 4
b = 7
c = 10
# calcola la media e stampala
```

<Solution>

```py live readonly
a = 4
b = 7
c = 10
media = (a + b + c) / 3
print(media)
```

</Solution>

</Exercise>
